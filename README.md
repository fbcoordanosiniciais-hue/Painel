# Painel
Office Dashboard: Agenda & Weather

Painel web desenvolvido para exibição em TVs 16:9

# Funcionalidades
Agenda Sincronizada: Exibição em tempo real e demandas.

Bing Wallpaper: Fundo de tela dinâmico atualizado diariamente via API do Bing.

Clima (Open-Meteo): Api gratis usada para disponibilizar o clima em tempo real.

Layout Responsivo: Tamanho dos objetos em cena editavel.

# Tecnologias e Lógica
Stack: HTML5, JavaScript (ES6+) e Tailwind CSS para estilização rápida.

Consumo de APIs: Uso de Fetch API para captura de dados meteorológicos e URLs de imagem.

Integração de Dados: O painel serve como interface de saída para fluxos de trabalho baseados em planilhas e formulários.

# Como Usar
Configure suas coordenadas geográficas no script de clima.

Vincule a URL da sua agenda no container principal.

Execute em modo tela cheia no monitor desejado.
